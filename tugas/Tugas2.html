Keterangan Tipe Data#

  • Kolom “Jenis Kelamin” merupakan tipe data biner asimetris karena nilainya hanya berupa laki-laki dan perempuan

  • Kolom “Usia” merupakan tipe data numerik karena nilainya berbentuk angka yang menggambarkan usia seseorang

  • Kolom “Status” merupakan tipe data merupakan tipe data kategrical/nominal karena terdiri dari kategori atau label yang hanya memiliki nilai atau nama tanpa memiliki urutan atau peringkat tertentu

  • Kolom “Riwayat Pendidikan” merupakan tipe data ordinal karena terdiri dari kategori atau label yang memiliki urutan atau peringkat

  • Kolom “Pengalaman Kerja” merupakan tipe data biner dikarenakan memiliki dua nilai

Code untuk menggabungkan atribut yang berbeda ke dalam matriks ketidaksamaan tunggal#

X = df[['Jenis Kelamin', 'Usia', 'Status', 'Riwayat Pendidikan', 'Pengalaman Kerja']].copy()
X['Jenis Kelamin'] = pd.Categorical(X['Jenis Kelamin']).codes
X['Status'] = pd.Categorical(X['Status']).codes
X['Riwayat Pendidikan'] = pd.Categorical(X['Riwayat Pendidikan']).codes
X['Pengalaman Kerja'] = pd.Categorical(X['Pengalaman Kerja'], categories=['SMP', 'SMA', 'D1', 'D3', 'S1'], ordered=True).codes

Code untuk menghitung matriks ketidaksamaan dengan skala interval antara [0,0,1.0]#

dist_matrix = pairwise_distances(X, metric='manhattan') / max(pairwise_distances(X, metric='manhattan').flatten())

Code untuk menampilkan matriks ketidaksamaan#
